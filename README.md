# NextJS 13 + TailwindCSS Minimalist Starter Project

## Technology

- NextJS v13.3.0
- TailwindCSS v3.3.1
- `tailwind-prettier-plugin` for automatic class sorting

## Using the starter

- Fork the project
- Clone to your local environment

- Install Packages
```bash
npm install
```

- Run The Development Server
```bash
npm run dev
```
